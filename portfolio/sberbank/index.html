Приложение «Sber QR» – геолокационная игра по типу «Ingress» и «Pokemon Go». С её помощью можно взаимодействовать с реальными компаниями на карте: покупка, аренда, продажа компаний.

Цель

Увеличить популярность безналичных платежей.

Задача

Научить людей через игру совершать платежи с помощью QR-кодов.

Функционал

Просмотр списка всех компаний-партнеров Сбера на карте.

Возможность просматривать карточку компании-партнера, находясь рядом с точкой.

Возможность арендовать часть компании, купить акции на определенное время.

Дополнительно
  • - продлить время аренды
  • - продать купленные акции
  • - получить дивиденды с арендованных акции компании
  • - чем больше игроков покупают одну компанию, тем она выше в цене
  • - повышение уровня в процессе игры благодаря полученному опыту
  • - уровень влияет на условные финансовые возможности игрока
